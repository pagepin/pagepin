Redesign D — the keyboard cockpit, refined after a full-flow walkthrough. The review rail is now a FLOATING DRAWER over the page's right edge instead of a column that reserves width, so the hosted page renders at its true width with zero reflow (what the reviewer sees is what visitors see). Comment creation is mouse-first — hovering any element shows a "+ comment" affordance, and pins are clickable — while the j/k · e · r · c · ⌘K grammar stays as an accelerator. A focused resolved thread reads clearly as resolved (Reopen), and the command palette starts closed so first paint is the calm resting state, not a modal. Collapse the drawer (\\) and the page is fully unobstructed.
